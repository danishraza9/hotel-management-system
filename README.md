Hotel Management System (CCP Project)

This project is a fully functional hotel management system developed in Java.  
It enables the management of hotels, rooms, and bookings while demonstrating object-oriented principles, SOLID design, defensive programming, and unit testing.

Key Features:
Create and manage hotels and rooms
Check room availability for specific dates
Filter rooms by type and price
Create, cancel, and query bookings
Calculate room occupancy rates and average room prices
Handle invalid inputs and errors using custom exceptions
Comprehensive JUnit 5 tests covering normal, boundary, and invalid scenarios
Main application demonstrating all use cases
